
This package includes Yifeng Li's implementations of capsule deep generative models. New developments will be gradually added.

1. Implementations in Python 3.6:

capsule_rbm.py: Class of capsule restricted Boltzmann machine (capsule RBM).
capsule_hm.py: Class of capsule Helmholtz machine. (capsule HM).


2. Examples:

main_test_ExpCRBM_MNIST.py: Capsule RBM on MNIST.
main_test_ExpCRBM_FASHIONMNIST.py: Capsule RBM on Fashion-MNIST.
main_test_ExpCHM_MNIST.py: Capsule HM on MNIST.
main_test_ExpCHM_FASHIONMNIST.py: Capsule HM on Fashion-MNIST.


3. Data:

MNIST: given in ./data/MNIST.
Fashion-MNIST: if missing, download the csv files from https://www.kaggle.com/zalando-research/fashionmnist, then added them to ./data/FASHIONMNIST.


4. Citations:

[1] Yifeng Li and Xiaodan Zhu. Capsule generative models. International Conference on Artificial Neural Networks (ICANN), under review, 2019.
